# MCP ハブ群の守備範囲と関係性: ntfy-hub-mcp, agent-hub-mcp, gistpad-mcp

本プロジェクトでは、AI エージェントの能力を拡張し、より高度な協調作業や人間とのインタラクションを可能にするために、複数の MCP (Model Context Protocol) サーバーの連携を構想しています。それぞれの MCP サーバーは異なる守備範囲を持ち、連携することで強力なエコシステムを形成します。

## 1. ntfy-hub-mcp (リアルタイム通知・指示チャネル)

*   **守備範囲**:
    *   **エージェントから人間へのリアルタイムな非同期通知**。
    *   **人間からエージェントへの簡易的な非同期指示**。
    *   PC の前にいなくてもスマートフォンなどへ通知が届くため、タイムリーな介入や承認を促す用途に特化。
*   **特徴**:
    *   軽量、高速。
    *   プッシュ通知に最適。
    *   緊急性の高い情報伝達、または人間の承認/選択待ち。
*   **関係性**:
    *   **`agent-hub-mcp` からの利用**: `agent-hub-mcp` の Orchestrator (管理 AI) が掲示板で重大な状況を検知したり、人間の介入が必須と判断した場合に、`ntfy-hub-mcp` を通じてユーザーに「掲示板を確認してください」「承認が必要です」といった通知を送る。
    *   **`gistpad-mcp` との連携**: `gistpad-mcp` に保存されたタスクリストの変更や、共有知識ベースの重要な更新があった際に、`ntfy-hub-mcp` を通じて関係者に通知する。

## 2. agent-hub-mcp (プロジェクト固有の協調作業ハブ)

*   **守備範囲**:
    *   **特定のプロジェクト内での AI エージェント間の詳細な協調作業と議論の場**。
    *   **エージェント間の進捗報告、問題提起、解決策の検討**。
    *   プロジェクトのコンテキストに密着した情報管理。
    *   人間がエージェント間のやり取りを俯瞰し、TUI (Terminal User Interface) を介して介入する場。
*   **特徴**:
    *   Go 言語、SQLite データベースによる永続的なログと状態管理。
    *   Orchestrator (管理 AI) による自律的なエージェント間調整。
    *   詳細な議論や決定プロセスの記録。
*   **関係性**:
    *   **`ntfy-hub-mcp` との連携**: 上述の通り、`agent-hub-mcp` は必要に応じて `ntfy-hub-mcp` を「通知ツール」として利用します。
    *   **`gistpad-mcp` との連携**: `agent-hub-mcp` での議論や成果が、プロジェクト横断的な知見として価値を持つと判断された場合に、`gistpad-mcp` に保存される共有知識ベースへ反映させる。

## 3. gistpad-mcp (プロジェクト横断的な共有知識ベース)

*   **守備範囲**:
    *   **複数のプロジェクトやエージェント間で共有すべき汎用的な知見、スニペット、ガイドライン、ベストプラクティスなどの管理**。
    *   Git リポジトリにコミットするほどではないが、永続化・共有したい情報の集約。
    *   エージェントが利用できる「外部脳」としての役割。
*   **特徴**:
    *   GitHub Gist をバックエンドとするため、バージョニングと共有が容易。
    *   コードスニペット、プロンプト例、Tips、FAQ などの蓄積。
*   **関係性**:
    *   **`ntfy-hub-mcp` との連携**: `gistpad-mcp` 上の重要な知見の更新や追加を、`ntfy-hub-mcp` を通じて関係者に通知する。
    *   **`agent-hub-mcp` との連携**: `agent-hub-mcp` で作業するエージェントが、共通の知識や解決策を参照するために `gistpad-mcp` から情報を取得する。また、`agent-hub-mcp` で得られた新しい知見を `gistpad-mcp` に追加する。

## 結論: シナジー効果

これら三つの MCP サーバーは、それぞれ異なる目的と守備範囲を持ちながらも、互いに連携することで AI エージェントの自律性、協調性、そして人間との効果的なインタラクションを飛躍的に向上させることができます。

-   **`gistpad-mcp`**: エージェントの**知識共有と学習**を促進。
-   **`agent-hub-mcp`**: エージェントの**協調作業とプロジェクト管理**を最適化。
-   **`ntfy-hub-mcp`**: エージェントと人間の**リアルタイムなコミュニケーションと介入**を可能にする。

これにより、AI エージェントはより賢く、より自律的に、そしてより安全に、複雑なタスクをこなせるようになるでしょう。
